## Fixed decisions (no remaining choices)
Ingress: cloud LB → Envoy Gateway (Gateway API) for control-plane-api only; TLS terminated at gateway, mesh mTLS inside. Certificates: cert-manager, Let's Encrypt for public, private CA (mesh) via Linkerd identity, 24h rotation. GPU scheduling (DP): collector DaemonSet tolerates all GPU-node taints; bench-runner Jobs pin `nvidia.com/gpu: 1`, node anti-noisy selector `nydux.ai/bench=true`. Secrets: External Secrets Operator → cloud secret manager; zero secrets in values files (CI grep gate). Multi-region: profiles per ECD-001 infra/envs; DR pairing prod-ap-south-1 ↔ dr per RFC-001 A.7.
