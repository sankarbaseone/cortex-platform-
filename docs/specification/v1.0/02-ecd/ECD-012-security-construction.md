**Level:** 2 · Extends RFC-009 (frozen). **Artifacts are the spec.**
- `artifacts/security/threat-model.md` — STRIDE × 6 trust boundaries, 15 threats each mapped to an implemented mitigation artifact + residual rating; CI-tested abuse-case list.
- `artifacts/security/opa/` — complete Rego: `rec_guard` (SoD + token + risk limit, with full opa test suite), `toolchain_gate` (approval + fleet-CRI gate), `agent_tool_guard` (write-class requires human token, RFC-008 H.5). - Audit action catalog (closed): tenant.create|freeze, user.invite|role_change, apikey.create|revoke, toolchain.register|approve|revoke, rec.create|approve|reject|apply|verify|rollback, baseline.create|reanchor, policy.create|update|delete|decision, deploy.trigger|rollback, agent.tool_call, export.evidence, dr.promote. Additive amendment recorded in kms module README + kms-layout — no frozen contract changed (RFC-009 named the key class, not the curve-per-provider).
